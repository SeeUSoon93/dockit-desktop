# Dockit Desktop

Dockit 웹앱의 데스크톱 래퍼입니다.

## 설치

```bash
npm install
```

## 실행

```bash
npm start
```

## 빌드

```bash
# Windows
npm run build:win

# macOS
npm run build:mac

# Linux
npm run build:linux
```

## 아이콘

`icons/` 폴더에 아이콘 파일 추가:
- `icon.ico` - Windows용 (256x256 권장)
- `icon.icns` - macOS용
- `icon.png` - Linux용 (512x512 권장)
